In my astronomy laboratory, a team and I completed a research project about the Iris Nebula. This project included taking several high resolution long exposure pictures of the Iris Nebula utilizing an 8" Meade telescope, reducing data using Unix and IDL to make a composite RGB image of the Nebula in colorful detail using data processing and analysis.
